Features:
User Authentication: Secure user registration and login using Passport.js with local strategy.

Image Uploads: Users can upload images along with captions, and the uploaded images are associated with their profiles.

Profile Page: Each user has a personalized profile page displaying their uploaded images in a visually appealing layout.

Feed: A feed page where users can view images uploaded by themselves and others.

Technologies Used:
Node.js: Backend JavaScript runtime.
Express: Web application framework for Node.js.
Passport.js: Authentication middleware for Node.js.
MongoDB: NoSQL database for storing user and image data.
Mongoose: MongoDB object modeling for Node.js.
EJS: Templating engine for generating HTML markup.
